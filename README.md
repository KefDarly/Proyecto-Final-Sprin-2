algo mas # 🚛 PetroMapi – Sistema de Monitoreo de Carga Ancha (Versión Final)

Sistema web completo para la gestión y monitoreo en tiempo real de flota de carga pesada de **PetroMapi Caminos S.A.C.**  
Desarrollado como proyecto final del Sprint 3, aplicando **Design Thinking** y asistido con **IA generativa**.

## 📌 Descripción

El sistema centraliza el control logístico, permitiendo:

- Monitoreo en vivo del estado de vehículos.
- Registro y alerta temprana de incidencias mecánicas/climáticas.
- Auditoría de consumo de combustible por ruta.
- Programación de mantenimientos preventivos.
- Gestión completa de flota (CRUD vehículos).

## 👥 Roles de usuario

| Rol        | Acceso principal                          |
|------------|---------------------------------------------|
| Admin      | Gestión total: usuarios, flota, reportes    |
| Operador   | Registro de incidencias y monitoreo básico  |
| Supervisor | Dashboards, costos y auditoría de combustible |
